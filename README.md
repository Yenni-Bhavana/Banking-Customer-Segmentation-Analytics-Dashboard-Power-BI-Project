# Banking-Customer-Segmentation-Analytics-Dashboard-Power-BI-Project
The objective of this project is to transform raw banking data into meaningful insights that help financial institutions understand customer behavior and financial activity.

# Project Overview
This project presents an interactive Banking Analytics Dashboard built using Power BI to analyze customer demographics, financial behavior, and transaction activity.
The objective of this project is to help financial institutions gain insights into:
* Customer segmentation
* Transaction patterns
* Customer demographics
* High-value customer identification
* Financial distribution across segments

The project contains two interactive dashboards:
1️. Bank Customer Segmentation Dashboard                                                     
2️. Transaction Analytics Dashboard                                                                        
These dashboards together provide a complete view of customer distribution, financial strength, and transaction behavior.

## Dashboard 1: Bank Customer Segmentation
### Objective

The Bank Customer Segmentation Dashboard focuses on understanding how customers are distributed across different segments based on balance, age group, gender, and financial value.This helps banks identify high-value customers and demographic patterns.

## Key KPIs
* 119.82 Billion Total Balance                                      
* 880K Total Customers
* 1M Total Transactions
* 1.57K Average Transaction Amount
* 173K High Value Customers
These KPIs provide a quick overview of the financial scale and customer base of the bank.

## Dashboard 2: Transaction Analytics
### Objective

The Transaction Analytics Dashboard focuses on analyzing transaction activity, customer spending patterns, and financial engagement.                              
This dashboard helps banks track transaction trends, top customers, and segment-level activity.

## Key KPIs
* 1M Total Transactions
* 880K Total Customers
* 1.63 Billion Total Transaction Amount
* 114.9K Average Balance
* 1.18 Transactions per Customer

### Key Insights:
* Premium customers hold the highest total balance (~91.76 bn) even though their count is lower, making them the most valuable segment for the bank.
* Low value customers are the largest group (~372K customers) and generate the highest number of transactions (~398K).
* The 36–50 age group has the highest number of customers (~0.54M), showing that this age group forms the core customer base.
* Male customers dominate the customer base (~71%) and contribute to the majority of transactions.
* A small group of top customers contributes a large share of the total transaction amount, highlighting the importance of high-value customers.
